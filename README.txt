NWP Slideshow is a customizable slideshow you can easily display on you website

== Description ==

NWP Slideshow is a WordPress plugin to display and customize a slideshow on your website. The slideshow is customizable in the back-office, you can edit content, transitions effects, the size, the transition speed, and more...

The plugin is for now available in english and french. Contact me if you want to help on the translation ;).

== Installation ==

1. Upload nwp-slideshow directory to the '/wp-content/plugins/' directory.
2. Set permissions tp './nwp-slideshow/images/slides/' directory as PHP can write into.
3. Enjoy it.
